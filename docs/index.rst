Что это такое
=============

Django EDW -- это пакет для Django, который систематизирует объекты в системе:

- Делает сложные семантические выборки
- Сортирует и группирует экземпляры разных моделей по общим признакам
- Генерирует REST API


Как это работает
================

Главная часть EDW -- это дерево терминов. Термин -- это свойство, которое может быть у объекта:

    Пример 1. Если пользователь администратор, значит у него есть свойство "Администратор". У простого пользователя такого своства нет.

    Пример 2. Публикация на сайте может быть статьёй, новостью, фоторепортажем, интервью -- всё это термины, которые могут у неё быть. 

Дерево терминов выглядит так:

.. image:: images/create-term-tree.png

Всё что вложено в "Объект" -- это термины, которые система создает сама, изучив модели. "Тематика", "Категория" и всё что в них вложено -- это термины, которые добавил администратор сайта.
