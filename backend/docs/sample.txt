First terminal:
go to your workdir
> git clone https://bitbucket.org/info-labs/django-edw.git
> cd django-edw
> virtualenv env
> source env/bin/activate
> pip install -r backend/requirements.txt
> cd ./backend/
> npm install
> bower install --allow-root

# if you use mysql database
> export EDW_DB_ENGINE=mysql
# for sqlite3
> export EDW_DB_ENGINE=sqlite3
> python manage.py runserver 0.0.0.0:8000 --insecure

Second terminal:
go to your workdir
> cd ./django-edw/frontend/
> npm install
> npm start
